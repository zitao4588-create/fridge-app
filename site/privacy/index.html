隐私政策

冰箱小雷达只围绕冰箱食材库存管理保存必要数据。当前个人主体稳定版不默认上传第三方 AI，不开放拍照识别、AI 菜谱、支付会员或社区功能。

收集什么

  • 当前微信用户 openid，用于数据隔离。
  • 用户主动录入的食材名称、品类、数量、分区、生产日期、保质期、过期日期和备注。
  • 订阅提醒授权结果和提醒发送记录，用于避免重复提醒。

不收集什么

  • 不主动读取通讯录。
  • 不默认读取相册。
  • 不默认把食材清单发送给第三方 AI。
  • 不出售用户数据。

数据用途

  • 展示冰箱库存和分区。
  • 计算临期、过期和到期日历。
  • 通过本地规则计算开饭雷达。
  • 发送用户授权后的到期提醒。

数据存储

食材数据存储在微信云开发 CloudBase 云数据库中，按当前微信用户 openid 隔离。小程序前端不保存第三方服务商 API key。

用户权利

用户可以在小程序中查看、编辑、删除单条食材，也可以通过首页底部的二次确认入口清空全部食材记录。

联系方式

如需处理隐私相关问题，请通过微信小程序后台配置的开发者联系方式或公开品牌页后续补充的联系邮箱联系开发者。

本页面为公开 HTML 隐私说明，正式上线后应与微信小程序后台隐私保护指引保持一致。
